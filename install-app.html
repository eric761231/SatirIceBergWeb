安裝 Obyssey App 到手機

iOS / iPad 安裝教學

  • 1. 使用 Safari 掃描上方 QR Code 或直接開啟 Obyssey 網站
  • 2. 點擊頁面上的「安裝App」按鈕
  • 3. 若未顯示安裝提示，請點選 Safari 下方「分享」按鈕 📤
  • 4. 選擇「加入主畫面」(Add to Home Screen)
  • 5. 點擊「加入」即可完成安裝，桌面會出現 Obyssey App 圖示

Android 安裝教學

  • 1. 使用 Chrome 掃描上方 QR Code 或直接開啟 Obyssey 網站
  • 2. 點擊頁面上的「安裝App」按鈕
  • 3. 若未顯示安裝提示，請點選瀏覽器選單「安裝 App」
  • 4. 按照提示完成安裝，桌面會出現 Obyssey App 圖示
如未顯示安裝提示，請確認瀏覽器版本或改用 Safari/Chrome。
