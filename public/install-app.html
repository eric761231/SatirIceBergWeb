📱 安裝冥想音樂應用程式

將應用程式安裝到手機主頁面，享受更好的使用體驗

🎯 為什麼要安裝應用程式？

安裝後，您可以：

  • 在手機主頁面直接開啟應用程式
  • 接收冥想提醒通知，即使應用程式未開啟
  • 享受離線播放功能
  • 獲得原生應用程式般的體驗

📱 安裝說明

Android 手機安裝步驟：

  1. 使用 Chrome 瀏覽器開啟冥想音樂頁面
  2. 點擊瀏覽器選單（三個點）
  3. 選擇「安裝應用程式」或「加入主畫面」
  4. 點擊「安裝」確認
  5. 應用程式將出現在主頁面上

iPhone/iPad 安裝步驟：

  1. 使用 Safari 瀏覽器開啟冥想音樂頁面
  2. 點擊底部的「分享」按鈕
  3. 選擇「加入主畫面」
  4. 點擊「加入」確認
  5. 應用程式將出現在主頁面上

桌面版安裝步驟：

  1. 使用支援 PWA 的瀏覽器（Chrome、Edge、Firefox）
  2. 在網址列右側會出現安裝圖標
  3. 點擊安裝圖標
  4. 選擇「安裝」確認
  5. 應用程式將出現在開始選單和桌面

✨ 安裝後的功能

  • 手機主頁面直接開啟
  • 背景推送通知提醒
  • 離線音樂播放
  • 全螢幕沉浸體驗
  • 快速存取播放清單
  • 自動更新和同步
